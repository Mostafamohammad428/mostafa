# نظام ERP المتكامل - نظام محاسبي شامل لجميع القطاعات

## نظرة عامة

نظام ERP (تخطيط موارد المؤسسة) المتكامل هو حل شامل لإدارة جميع عمليات الشركة من المحاسبة والمبيعات والمشتريات والمخزون والموارد البشرية والمشاريع. تم تطوير النظام باستخدام أحدث التقنيات لضمان الأداء العالي والأمان.

## الميزات الرئيسية

### 🏢 لوحة التحكم الشاملة
- إحصائيات مالية شاملة (الإيرادات، المصروفات، الأرباح)
- مؤشرات الأداء الرئيسية لجميع الأقسام
- تتبع الأنشطة الحديثة والطلبات
- تحليلات بصرية للبيانات

### 💰 نظام المحاسبة
- دليل حسابات متكامل
- قيود يومية وترحيل للحسابات
- تتبع الحسابات المدينة والدائنة
- ربط تلقائي بجميع العمليات المالية

### 🛒 إدارة المبيعات
- إنشاء وإدارة طلبات المبيعات
- متابعة حالة الطلبات
- ربط العملاء بالطلبات
- حساب الضرائب والخصومات

### 📦 إدارة المشتريات
- طلبات الشراء والمتابعة
- إدارة الموردين
- ربط المشتريات بالمخزون
- تتبع التكاليف والفواتير

### 📋 إدارة المخزون
- تتبع المنتجات والكميات
- تنبيهات المخزون المنخفض
- حركات المخزون (دخول/خروج/تسوية)
- تقييم المخزون بالتكلفة

### 👥 إدارة الموارد البشرية
- ملفات الموظفين الشاملة
- إدارة الأقسام والمناصب
- نظام الرواتب والبدلات
- تتبع حالة الموظفين

### 🏗️ إدارة المشاريع
- تخطيط وتتبع المشاريع
- إدارة الميزانيات والتكاليف
- ربط المشاريع بالعملاء
- تقارير تقدم المشاريع

### 👤 إدارة العملاء (CRM)
- قاعدة بيانات العملاء
- تتبع حدود الائتمان والأرصدة
- سجل المعاملات مع العملاء
- تحليل سلوك العملاء

### 🏪 إدارة الموردين
- قاعدة بيانات الموردين
- تصنيف الموردين حسب الفئة
- تقييم أداء الموردين
- سجل المشتريات والمدفوعات

### 📈 التقارير والتحليلات
- تقرير الأرباح والخسائر
- الميزانية العمومية
- تقرير التدفق النقدي
- تقارير المبيعات والمشتريات
- تقارير المخزون والموارد البشرية

## التقنيات المستخدمة

### Backend (الخادم)
- **FastAPI**: إطار عمل Python حديث وسريع
- **MongoDB**: قاعدة بيانات NoSQL مرنة
- **Motor**: محرك MongoDB غير متزامن
- **Pydantic**: التحقق من صحة البيانات
- **JWT**: المصادقة الآمنة
- **Bcrypt**: تشفير كلمات المرور

### Frontend (الواجهة)
- **React 19**: مكتبة JavaScript للواجهات
- **Tailwind CSS**: إطار عمل CSS للتصميم
- **Axios**: مكتبة HTTP للاتصال بالخادم
- **Context API**: إدارة الحالة العامة

## متطلبات التشغيل

### الخادم
- Python 3.8+
- MongoDB 4.4+
- pip (مدير حزم Python)

### الواجهة
- Node.js 16+
- npm أو yarn
- متصفح حديث

## التثبيت والتشغيل

### 1. إعداد قاعدة البيانات
```bash
# تثبيت MongoDB
# Ubuntu/Debian
sudo apt-get install mongodb

# macOS
brew install mongodb-community

# تشغيل MongoDB
sudo systemctl start mongod
# أو
brew services start mongodb-community
```

### 2. إعداد الخادم (Backend)
```bash
# الانتقال لمجلد الخادم
cd backend

# إنشاء بيئة افتراضية
python -m venv venv

# تفعيل البيئة الافتراضية
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

# تثبيت المتطلبات
pip install -r requirements.txt

# تشغيل الخادم
uvicorn server:app --reload --host 0.0.0.0 --port 8000
```

### 3. إعداد الواجهة (Frontend)
```bash
# الانتقال لمجلد الواجهة
cd frontend

# تثبيت المتطلبات
npm install
# أو
yarn install

# تشغيل الواجهة
npm start
# أو
yarn start
```

## الاستخدام

### 1. إنشاء حساب جديد
- افتح المتصفح على `http://localhost:3000`
- اختر "تسجيل جديد"
- املأ البيانات المطلوبة
- اختر الدور المناسب (مدير، محاسب، مبيعات، إلخ)

### 2. تسجيل الدخول
- استخدم اسم المستخدم وكلمة المرور
- ستنتقل للوحة التحكم الرئيسية

### 3. إعداد النظام الأولي
1. **إنشاء دليل الحسابات**:
   - انتقل لقسم "المحاسبة"
   - أضف الحسابات الأساسية (النقدية، البنك، العملاء، الموردين، إلخ)

2. **إضافة العملاء والموردين**:
   - انتقل لقسم "العملاء" و "الموردين"
   - أضف بيانات العملاء والموردين

3. **إعداد المنتجات**:
   - انتقل لقسم "المخزون"
   - أضف المنتجات مع الأسعار والكميات

4. **إضافة الموظفين**:
   - انتقل لقسم "الموارد البشرية"
   - أضف بيانات الموظفين والأقسام

### 4. العمليات اليومية
- **المبيعات**: إنشاء طلبات مبيعات جديدة
- **المشتريات**: إنشاء طلبات شراء
- **المحاسبة**: تسجيل القيود المحاسبية
- **المخزون**: تسجيل حركات المخزون
- **المشاريع**: متابعة تقدم المشاريع

## النشر في الإنتاج

### 1. إعداد البيئة
```bash
# تحديث متغيرات البيئة
# backend/.env
MONGO_URL=mongodb://your-production-db
SECRET_KEY=your-super-secure-secret-key
DB_NAME=erp_production

# frontend/.env
REACT_APP_BACKEND_URL=https://your-domain.com
```

### 2. بناء الواجهة
```bash
cd frontend
npm run build
```

### 3. نشر الخادم
```bash
# استخدام Gunicorn للإنتاج
pip install gunicorn
gunicorn server:app -w 4 -k uvicorn.workers.UvicornWorker
```
